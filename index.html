🛡️ Attenzione: file aperto in locale (file:///)

Quando la pagina è aperta direttamente dal computer, il browser non invia il "referer" a YouTube e alcuni video restituiscono l'Errore 153.

Soluzione definitiva: avvia la pagina da un piccolo server locale (es. python -m http.server nella cartella del file, poi apri http://localhost:8000). Oppure prova comunque: molti video partono lo stesso e, se uno è bloccato, l'app passa automaticamente al successivo.
